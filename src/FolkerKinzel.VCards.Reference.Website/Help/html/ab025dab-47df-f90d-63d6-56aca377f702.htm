      XmlBuilder Methods

      Methods

      Add Adds an XmlProperty instance, which is newly initialized using the specified arguments, to the Xmls property.
      Clear Sets the Xmls property to null.
      Edit(FuncIEnumerableXmlProperty, IEnumerableXmlProperty) Edits the content of the Xmls property with a delegate.
      EditTData(FuncIEnumerableXmlProperty, TData, IEnumerableXmlProperty, TData) Edits the content of the Xmls property with a delegate and allows to pass data to this delegate.
      GetTypeGets the Type of the current instance.
      (Inherited from Object)
      Remove Removes XmlProperty objects that match a specified predicate from the Xmls property.
      SetIndexes Sets the Index properties of the items in in the Xmls property ascending depending on their position in that collection and allows to specify whether to skip empty items in that process.
      SetPreferences Sets the Preference properties of the items in the Xmls property depending on their position in that collection and allows to specify whether to skip empty items in that process. (The first item gets the highest preference 1.)
      UnsetIndexes Resets the Index properties of the items in in the Xmls property to null.
      UnsetPreferences Resets the Preference properties of the items in in the Xmls property to the lowest value (100).
