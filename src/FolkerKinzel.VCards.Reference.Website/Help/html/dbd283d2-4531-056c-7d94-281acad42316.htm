      FolkerKinzel.VCards.Models.PropertyParts Namespace

      Classes of the vCard data model that your application can't instantiate. Therefore it won't be necessary to expose this namespace in your code.

      Classes

      AddressEncapsulates information about a postal delivery address.
      DataPropertyValue Encapsulates external data in a VCard. This can be either an array of Bytes, an absolute Uri, or a String.
      DateAndOrTime Encapsulates data that describes a date and/or a time. This can be a DateOnly value, a DateTimeOffset value, a TimeOnly value, or a String.
      GenderInfoEncapsulates information to specify the components of the gender and gender identity of the object the VCard represents.
      NameEncapsulates information about the name of the person the VCard represents.
      ParameterSectionEncapsulates the information about the parameters of a vCard property.
      Relation Encapsulates the data that describes a relation with a person or organization. This can be a String, a VCard, a Guid, or an absolute Uri.
