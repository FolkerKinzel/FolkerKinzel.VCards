      uuid  Guid
      A Guid that refers to the vCard of the person or organization via its ID property (UID).
      relationType  NullableRel  (Optional)
      Standardized description of the relationship with the person or organization to whose vCard the uuid refers. RelationType of the returned instance will be set to this value.
      group  String  (Optional)
      Identifier of the group of VCardProperty objects, which the VCardProperty should belong to, or null to indicate that the VCardProperty does not belong to any group.

      Return Value

      RelationProperty
      The newly created RelationProperty instance.

      Remarks

      Normally this method will not be used in own code. Use FromVCard(VCard, NullableRel, String) instead if you own the VCard. When serializing VCF the VCards will be automatically replaced by their Guids and when parsing VCF the Guids will be automatically dereferenced to their corresponding VCards if these VCards are available.
