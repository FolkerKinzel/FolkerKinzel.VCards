      Remove Removes OrgProperty objects that match a specified predicate from the Organizations property.
      SetIndexes Sets the Index properties of the items in in the Organizations property ascending depending on their position in that collection and allows to specify whether to skip empty items in that process.
      SetPreferences Sets the Preference properties of the items in the Organizations property depending on their position in that collection and allows to specify whether to skip empty items in that process. (The first item gets the highest preference 1.)
      UnsetIndexes Resets the Index properties of the items in in the Organizations property to null.
      UnsetPreferences Resets the Preference properties of the items in in the Organizations property to the lowest value (100).
